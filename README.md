# VibeCast

VibeCast is a professional-grade Walkie-Talkie Flutter application featuring a Nothing OS-inspired aesthetic. 

## Features
- **Push-To-Talk (PTT)**: Real-time walkie-talkie communication.
- **Dynamic Waveform**: Interactive PTT waveform animation that reacts in real-time to microphone input.
- **Group Communication**: Group creation with global state synchronization and persistence.
- **Ergonomic UI/UX**: Professional-grade, high-performance interface with a carefully designed layout for the PTT button.
- **Nothing OS Aesthetic**: Minimalist, clean, and modern design language.

## Architecture
This app follows a clean architecture pattern utilizing `flutter_bloc` for state management, `go_router` for navigation, and `dio` for robust networking.

## Getting Started

To run this project:
1. Clone the repository.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` on your preferred device or emulator.
